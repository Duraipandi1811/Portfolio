## ⚙ Installation & Setup

Clone repository:

```bash
git clone https://github.com/your-username/portfolio.git
```

Move into folder:

```bash
cd portfolio
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

Open browser:

```bash
http://localhost:5173
```

Build production version:

```bash
npm run build
```

Preview build:

```bash
npm run preview
```
