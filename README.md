## ⚙ Installation & Setup

Clone repository:

```bash
git clone https://github.com/Duraipandi1811/portfolio.git
```

Move into project:

```bash
cd portfolio
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

Open:

```bash
http://localhost:5173
```

Build production:

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```
